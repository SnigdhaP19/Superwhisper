Introduction

The most likely reason why you have been brought here is to probably see how a messaging mode can go beyond everyday texting.

A good way to demonstrate Superwhisperâ€™s messaging abilities is by using Slack.

You do not even need to type the username to ping!

Below is how a messaging mode can be set up for building slack mentions.

Steps

  • Open Superwhipser
  • Go to Mode
  • You may need to create a mode for this if this is first time using
  • Change the language model to GPT 3.5 Turbo for best effect
  • Add a language prompt (recommended to give it some slack ids so it can understand that it needs to ping using @)
  • Add examples if needed (using words like mention will really help)
  • Go to text replacements (do this if you cannot seem to debug some mistakes)
  • Add in what the word or phrase it is outputting in the left column
  • Add what you want in the right column
  • Start Recording
