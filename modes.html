Introduction

Superwhisper is not just your normal voice dictation app.

It can be used for a variety of tasks.

Such tasks are called Modes in Superwhisper.

They are created according to the different types of tasks that might require voice dictation over typing.

Below are the six modes that Superwhisper offers.

Whatâ€™s even more great is that you can create your own mode with the setup of the existing modes!


Messages

This is the mode that is created for using voice dictation to create a text message or SMS.

Since this software is available on a macOS device, this mode can be used to send iMessages or other text messages.

It is not only limited to your everyday texting.

You can also use this mode for sending an outreach on Linkedin (add a link) or messaging on slack (add a link)


Email

This is the mode that is created for using voice dictation to create emails or reply to them.

This is arguably the most simple mode to use because you do not need to specify input or output.

If you are new user of Superwhisper, then it is highly recommended that you try the email mode first.

By doing this you can also practice writing an AI model prompt (add a link) that is required for the other modes.


Note

This is the mode that is created for using voice dictation to create notes.

Usually Superwhisper users use their own voice to take down notes.

As already mentioned, Superwhisper is not an ordinary voice dictation software.

In this mode, you can specify how you want the notes to be formatted (add a link to medical chart).

You can also get Superwhisper to take another audio and make notes (add a link to Note taking)
